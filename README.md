# ElvUI_WishFlex

**ElvUI_WishFlex** 是一款专为魔兽世界（World of Warcraft）设计的 **ElvUI** 插件增强包。它通过一系列高度模块化的功能，优化了战斗视觉反馈、界面自动化隐藏以及沉浸式的剧情交互体验，旨在打造一个更整洁、更现代、更智能的玩家界面。

---

## ✨ 核心功能模块

### 🛡️ 战斗与视觉增强
* **AuraGlow / Glow**: 为重要光环、触发技能提供高度可配置的发光特效，确保关键信息不被遗漏。
* **SpellAlpha**: 动态调整技能按键的透明度，优化视觉焦点的清晰度。
* **StripeSkin**: 为 UI 元素注入统一的条纹纹理风格，提升整体界面的一致性与质感。

### 🤖 界面智能化 (SmartHide)
* **智能隐藏**: 根据玩家当前的战斗状态、目标选择或环境变化，自动隐藏或淡出非必要的 UI 元素，在非战斗状态下还你一个干净的屏幕。

### 📖 沉浸式体验 (Narrative)
* **叙事任务界面**: 彻底重绘任务与对话窗口，提供类似现代 RPG 游戏的沉浸式叙事体验，让任务阅读变得轻松有趣。

### 📊 核心监控
* **CooldownTracker / CooldownCustom**: 强大的冷却时间追踪系统，支持自定义重要技能的显示逻辑。
* **LustMonitor**: 专用的嗜血/英勇计时器，包含独特的视觉动画反馈。
* **ClassResource**: 针对各职业特有资源（如连击点、怒气等）的视觉美化与增强。

### 🛠️ 实用工具集
* **WishTargetAlert / RareAlert**: 进入战斗、获得目标或发现稀有精英时的醒目文字/声音警报。
* **WorldMarker**: 快速放置世界标记与光柱，简化团队协作流程。
* **KeyBinder**: 极其简便的快速按键绑定工具，告别繁琐的设置菜单。
* **ChatSetup**: 一键优化聊天框样式与布局。

---

## 🛠️ 安装要求

* **必须安装 [ElvUI](https://www.tukui.org/welcome.php)**：本插件作为 ElvUI 的扩展模块运行，需确保其已处于启用状态。

## 🚀 安装步骤

1.  下载最新的 [Release 压缩包](https://github.com/WishFlex/ElvUI_WishFlex/releases)。
2.  将解压后的 `ElvUI_WishFlex` 文件夹放入你的魔兽世界目录中：
    `_retail_/Interface/AddOns/`
3.  进入游戏，在 **ElvUI 设置界面** 中即可找到 **WishFlex** 的专属配置页。
